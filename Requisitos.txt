Enunciado
Se debe crear una página web que sólo use HTML y CSS. El tema, el
contenido, colores, etiquetas entre otros lo decide cada estudiante.
Cómo mínimo debe contener 10 etiquetas distintas en el html de su página
Web. Los estilos CSS deben ser específicos, como lo visto en clase. Para la
ubicación de los componentes en la Web, se debe usar Flexbox.

Rúbrica de evaluación

Obligatorias:

● No se debe usar la etiqueta &lt;div&gt;&lt;/div&gt;. Se cobrará 0.1 por cada uso
de dicha etiqueta.
● La página debe contener como mínimo 10 etiquetas distintas en su
construcción. Rebaja de 0.5 si no cumple
● El CSS debe contener la mayor especificidad posible. Debe contener
como mínimo 10 estilos que no sean .class ó #id. Rebaja de 0.5 si no
cumple.
● Uso de Flexbox para la distribución del espacio de la página. Rebaja
de 0.5 si no cumple.
● Un único archivo CSS (sin usar librerías) para los estilos de la página.
Rebaja de 0.5 si no cumple.
